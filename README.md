
ローカルでの動かし方
===================

前提
----

- Node.js（推奨: 18以上）がインストールされていること
- npm または pnpm が利用可能であること

インストールと開発サーバ起動（macOS / zsh）
------------------------------------------

1. リポジトリのルートに移動します。

```zsh
cd /Users/hiroshimatakara/Downloads/専門分野特化型ai記事生成アプリ
```

2. 依存関係をインストールします。

```zsh
npm install
# または pnpm を使う場合
# pnpm install
```

3. 開発サーバを起動します。

```zsh
npm run dev
# デフォルトでは Vite が起動し、通常は http://localhost:5173 でアクセスできます
```

ブラウザで確認
----------------

開発サーバ起動後にブラウザで以下にアクセスしてください。

- http://localhost:5173

（補足）本番ビルドとプレビュー
-----------------------------

必要に応じて本番ビルドとプレビューを行えます。

```zsh
npm run build
npm run preview
```

以上。ローカルでの起動方法のみを記載しています。
